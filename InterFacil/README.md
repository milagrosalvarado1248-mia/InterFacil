Descripción
InterFácil es una aplicación móvil desarrollada con React Native y Expo que permite calcular automáticamente el aumento de un alquiler utilizando los índices IPC publicados por el INDEC.

Funcionalidades
•	Pantalla de inicio con marca y accesos principales
•	Listado de índices IPC con mes y porcentaje
•	Detalle de cada índice con información completa
•	Formulario controlado para calcular el nuevo valor del alquiler
•	Consumo de datos mediante service (API pública o mock)
•	Manejo de estados: loading, empty, error y success

Instalación y ejecución
pnpm install
npx expo start
